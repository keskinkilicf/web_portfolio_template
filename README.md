## 🚀 Proje Hakkında
Tasarımda modern web trendlerinden biri olan **Glassmorphism (Cam Efekti)** kullanılmış olup, arayüz derinlikli, karanlık tema ve neon detaylarla desteklenmiştir. 

Amacım, kodlamadaki analitik mantığı, yaratıcı tasarımlarla birleştirerek kullanıcılara akıcı bir dijital deneyim sunmaktır.

## ✨ Öne Çıkan Özellikler
- **Modern UI/UX:** Yarı saydam kartlar, backdrop-filter kullanımları ve akıcı hover animasyonları.
- **Dinamik Proje Filtreleme:** Projeler sekmesinde Vanilla JS ile yazılmış, kategorilere göre yumuşak geçişli filtreleme sistemi.
- **Özel Bildirimler (Toast):** İletişim formunda sayfanın yenilenmesini engelleyerek çalışan, sağ üst köşeden kayarak gelen modern JS bildirim sistemi.
- **Tamamen Responsive:** CSS Grid ve Flexbox mimarisiyle cihaz bağımsız esnek görünüm.

## 🛠️ Yeteneklerim & Araçlar
Ağırlıklı olarak web teknolojileri, nesne tabanlı programlama ve dinamik içerik üretimi üzerine çalışıyorum:

* **Web Geliştirme:** HTML5, CSS3, JavaScript (ES6+)
* **Programlama Dilleri:** Java
* **Tasarım & Animasyon:** Adobe Animate (2D Karakter/Yürüme döngüsü animasyonları), Canva (Espor afişleri, Vektörel logolar, Jenerikler)
